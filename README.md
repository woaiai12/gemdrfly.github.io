## 🚀 快速开始

### 本地运行

1. 克隆仓库
```bash
git clone https://github.com/woaiai12/gemdrfly.github.io.git
cd gemdrfly.github.io
```

2. 在浏览器中打开 `index.html`

### 部署到 GitHub Pages

1. 修改文件
2. 运行同步脚本：
```bash
sync.bat
```

或使用自定义消息：
```bash
sync-with-message.bat "你的提交消息"
```
